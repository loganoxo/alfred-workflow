## 新建文件

> alfred
> > 输入 new，回车 : 在默认目录新建
>
> > 输入 new，command+回车 : 选择目录新建

> > 输入 文件名，回车 : 新建文件保留空格; 重复文件名末尾会加数字(扩展名之前)
>
> > 输入 文件名，command+回车 : 新建多个文件, 以空格为分隔符; 重复文件名末尾会加数字(扩展名之前)
